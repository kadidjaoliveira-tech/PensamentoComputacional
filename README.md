# Pensamento Computacional

## Atividades Práticas

Este repositório reúne atividades práticas desenvolvidas para a disciplina de **Pensamento Computacional**, com o objetivo de estimular habilidades como lógica, resolução de problemas, decomposição, reconhecimento de padrões e abstração.

## 📚 Sobre o Projeto

As atividades aqui presentes foram elaboradas para aplicar conceitos fundamentais do pensamento computacional de forma prática e acessível. Elas podem ser utilizadas tanto por estudantes quanto por educadores como material de apoio.

##  Objetivos

* Desenvolver o raciocínio lógico
* Incentivar a resolução estruturada de problemas
* Aplicar conceitos computacionais sem necessariamente usar programação
* Estimular o pensamento crítico e criativo

##  Conteúdo

O repositório pode incluir atividades como:

* Sequências lógicas
* Algoritmos simples
* Fluxogramas
* Jogos e desafios de lógica
* Problemas do cotidiano resolvidos com abordagem computacional

##  Como Utilizar

1. Escolha uma atividade
2. Leia atentamente a proposta
3. Tente resolver o problema passo a passo
4. Compare sua solução com sugestões (quando disponíveis)
5. Reflita sobre o processo utilizado

##  Requisitos

Não há requisitos técnicos obrigatórios. Algumas atividades podem exigir:

* Papel e caneta
* Ferramentas de desenho (para fluxogramas)
* Computador (opcional)
